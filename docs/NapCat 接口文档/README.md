## 通用响应格式

所有接口返回统一的 JSON 结构：

```json
{
    "status": "ok",
    "retcode": 0,
    "data": {},
    "message": "",
    "wording": "",
    "stream": "normal-action"
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `status` | string | `ok` 表示成功，`failed` 表示失败 |
| `retcode` | integer | `0` 成功；`1400` 参数错误；`1401` 权限不足；`1404` 资源不存在 |
| `data` | object/null | 业务数据，失败时为 `null` |
| `message` | string | 错误信息 |
| `wording` | string | 错误信息（用户友好描述） |
