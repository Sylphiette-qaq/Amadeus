# 频道接口

## POST 获取频道列表

POST /get_guild_list

获取当前帐号已加入的频道列表

> Body 请求参数

```json
{}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|

> 返回示例

> 业务响应

```json
{
    "status": "ok",
    "retcode": 0,
    "data": [
        {
            "guild_id": "123456",
            "guild_name": "测试频道"
        }
    ],
    "message": "",
    "wording": "",
    "stream": "normal-action"
}
```

```json
{

    "retcode": 1400,
    "data": null,
    "message": "请求参数错误或业务逻辑执行失败",
    "wording": "请求参数错误或业务逻辑执行失败",

}
```

```json
{
    "status": "failed",
```
